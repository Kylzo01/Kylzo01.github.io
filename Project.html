Project 1 - Arduino Project


Keep Talking and Nobody explodes
(which our game is inspired by)
The first project I undertook was in a group as a part of my computing science course at RGU. For this project we had to use a arduino and some atatchments to create a device that is a product that would be useful for people. We decided to create a bomb disposal game. The game will have differenct modules that will be required to be completed so that the "bomb" does not explode. The game will be created using an arduino, a couple breadboards, wires, buzzers, buttons, led's and will be coded using Java and Processing.
Our game was inspired by the VR game "Keep talking and nobody explodes". This game is also a bomb disposal game where one player reads out instruction to another player who has to dismantle a bomb.
